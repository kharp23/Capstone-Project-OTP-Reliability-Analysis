# Data Download Instructions

## Dataset Source
The raw datasets used in this project are publicly available and can be downloaded from the original Kaggle source for year 2020.

## Dataset Link:
(https://www.kaggle.com/datasets/pranavbadami/nj-transit-amtrak-nec-performance)

## Required Files
Download the following files:
* 2020_01.csv
* 2020_02.csv
* 2020_03.csv
* 2020_04.csv
* 2020_05.csv

## Access Requirements
A free Kaggle account may be required to access and download the dataset.

## Reproducing the Analysis
1. Download the dataset from the Kaggle link provided above.
2. Open the notebooks located in the notebooks folder.
3. Update the file paths in the notebooks if necessary to match the location of the downloaded files on your computer.
4. Run the notebooks in the order specified in the repository README.

## Notes
The raw datasets are not included in this GitHub repository because they exceed GitHub's recommended file size limits. The original data source is provided to ensure that the analysis can be independently reproduced using the same data.
